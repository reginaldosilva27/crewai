# Commands

## List containers
> docker ps -a

## Remove containers 
> docker rm xxx

## List docker images
> docker image ls

## Build an image 
> docker build -t fastapicrew .    

## Run a container image 
> docker run -d --name fastapidemocrew -p 80:80 fastapicrew   

## Install Streamlit
> pip install streamlit
>> pip install streamlit-lottie

## Run code
> streamlit run testeAPIStreamlit.py
